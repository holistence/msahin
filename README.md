# Modern Akademik Web Sitesi

Bu paket GitHub Pages üzerinde yayınlanabilecek tek sayfalık, modern ve animasyonlu bir akademik web sitesi içerir.

## Dosyalar
- `index.html`
- `css/style.css`
- `js/script.js`

## Kurulum
1. Bu dosyaları GitHub reponuza yükleyin.
2. GitHub > Settings > Pages bölümüne gidin.
3. `Deploy from a branch` seçin.
4. `main` ve `/root` ayarını seçin.
5. Site adresiniz birkaç dakika içinde aktif olur.

## Özelleştirilecek alanlar
- İsim ve ünvan
- Hakkımda metni
- Ders kartları
- Yayın listesi
- CV zaman çizelgesi
- İletişim ve akademik bağlantılar
